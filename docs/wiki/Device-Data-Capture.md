Treetracker app collects device data such as 

- GPS
- Steps count
- Gyroscope and Compass Heading

All of these data are sent to the cloud services at Greenstand to help with analysis and verification of the trees captured by the planters. The analysis is important in determining various aspects of tree tracking activities. Some of them are

- Tree updates (growth tracking) a good convergence of location data increases the confidence about the accuracy of tree location and in identifying and ascribing the tree's growth correctly.
- Compas heading and orientation data of the device would help in aiding the image processing of captured tree images

The document [Location Collection Details](https://github.com/Greenstand/Development-Overview/wiki/Android-v1.4-GPS-location-data-collection) 
details the motivation and requirements specified by the data analysis team at Greenstand.
