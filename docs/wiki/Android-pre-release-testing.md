**Testing**

* Login with the phone

    User should be able to use either number or email or both. Special characters should be permitted. 

    If logging in twice with the same phone number, the phone should skip the name/org page and go directly to the selfi image. 

* User should be able to change users. 

* Add data points.

    The app has a block for GPS accuracy. The user should be able to get GPS reception, "add a tree." Take a photo and upload the image. 

* Sync
The phone should be able to sync images. 

* Images should appear on the map at test.treetracker.org
