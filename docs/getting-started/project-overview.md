# Project Overview

Treetracker Android is a field data collection app for the Greenstand open source project.

The app captures tree growth data in the field and uploads verifiable records to Greenstand services for analysis and attribution workflows.

The product mission centers on climate restoration and creating income opportunities through verified tree-growing activity.
