# Device Data Capture

The app captures device telemetry to support analysis and verification:

- GPS
- Step count
- Gyroscope and compass heading

This data helps improve confidence in:

- Tree location convergence
- Distinguishing updates from new captures
- Image/context interpretation during verification

Reference: [Android v1.4 GPS location data collection](https://github.com/Greenstand/Development-Overview/wiki/Android-v1.4-GPS-location-data-collection)
